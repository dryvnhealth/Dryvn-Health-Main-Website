# DRYVN Health Website

Official website for **Dryvn Health LLC** — a Massachusetts-based healthcare staffing company built on compliance, integrity, and excellence.

## About Dryvn Health

Dryvn Health is a new kind of healthcare staffing agency focused on treating clinicians with respect, transparency, and elite benefits. We deliver higher-quality staff, faster fill times, and dramatically lower turnover because we treat our clinicians better than anyone else.

**Core Values:**
- Compliance-first with zero tolerance for non-compliance
- Integrity in every placement
- Clear communication and fighting for resolutions
- Adapting to each facility's unique needs without compromising quality
- Drive for excellence in everything we do

## Tech Stack

- HTML5 + Tailwind CSS
- Vanilla JavaScript
- Hosted on Vercel
- Domain: dryvnhealth.com

## How to Update the Website

1. Edit the `index.html` file
2. Commit and push to GitHub
3. Vercel automatically deploys the changes in ~30-60 seconds

## Deployment

This site is deployed via **Vercel** connected to this GitHub repository.

- Live site: https://dryvnhealth.com
- Vercel Dashboard: https://vercel.com
